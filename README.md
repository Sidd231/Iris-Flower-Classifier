# 🌸 Iris Flower Classifier

A Machine Learning project that classifies iris flowers into three species 
— Setosa, Versicolor, and Virginica — based on their petal and sepal measurements.

Built using **Python** and **scikit-learn**.

## 📊 Results

| Metric | Score |
|--------|-------|
| Accuracy | 100% |
| Algorithm | K-Nearest Neighbors (KNN) |
| Dataset Size | 150 samples |

## 🌺 The Three Species

| Species | Characteristics |
|---------|----------------|
| Setosa | Small petals, easily separable |
| Versicolor | Medium sized, overlaps with virginica |
| Virginica | Largest petals |

## 🛠️ Tech Stack

## 🚀 How to Run

1. Clone the repository
```bash
git clone https://github.com/Sidd231/iris-flower-classifier
cd iris-flower-classifier
```
2. Install dependencies
```bash
pip install scikit-learn pandas numpy matplotlib
```
3. Open `iris.ipynb` in VS Code and run all cells

## 📈 What the Model Does
1. Loads the built-in Iris dataset from scikit-learn
2. Visualizes species separation by petal size
3. Trains a KNN classifier with k=3
4. Achieves 100% accuracy on test data
5. Predicts the species of a brand new flower
